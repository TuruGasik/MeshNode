MeshNode Indonesia

Peta realtime node Meshtastic Indonesia

👋 Selamat datang di peta node Meshtastic Indonesia! Peta ini menampilkan node-node aktif yang terdeteksi melalui jaringan MQTT komunitas Indonesia.
Fitur
  • Menampilkan node yang mengirimkan posisi GPS valid ke server MQTT
  • Posisi harus tidak terenkripsi atau menggunakan kunci enkripsi default
  • Gunakan fitur pencarian untuk menemukan node berdasarkan ID atau nama
  • Arahkan kursor ke node (desktop) atau ketuk (mobile) untuk melihat info singkat
  • Klik node untuk melihat detail lengkap seperti telemetri dan koneksi antar node
FAQ
Bagaimana cara agar node saya muncul di peta?
Node kamu — atau node lain yang mendengar sinyalmu — harus melakukan uplink ke server MQTT kami. Paket posisi harus tidak terenkripsi atau menggunakan kunci enkripsi default. Jika menggunakan firmware v2.5 ke atas, pastikan opsi "OK to MQTT" sudah diaktifkan. Gunakan detail server MQTT di bawah untuk terhubung ke peta ini.
Server MQTT mana yang harus digunakan?
  • Address: mqtt.meshnode.id
  • Username: idmeshnode
  • Password: Mesh4all
  • Enkripsi: Aktif
  • JSON Output: Tidak
  • TLS: Aktif
  • Topic: msh/ID
Catatan: Disarankan menggunakan node terpisah yang terhubung ke internet khusus untuk uplink ke peta, agar node utamamu tetap berjalan di jaringan mesh lokal.
Bagaimana cara menghapus node saya dari peta?
Node yang tidak aktif selama lebih dari 1 hari akan otomatis disembunyikan dari tampilan peta, namun datanya tetap tersimpan di database. Untuk mencegah node muncul kembali, nonaktifkan pelaporan posisi di pengaturan node kamu. Alternatifnya, gunakan kunci enkripsi kustom agar data posisimu tidak dapat dibaca publik.
Mengapa node saya muncul di lokasi yang tidak tepat?
Peta ini secara sengaja menyamarkan posisi node yang menggunakan firmware v2.4 atau lebih lama — akurasi posisi dibatasi hingga 2 angka desimal saja. Perbarui firmware ke v2.5 ke atas untuk mendapatkan posisi yang lebih akurat di peta.
