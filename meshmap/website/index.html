MeshNode Indonesia

Peta realtime node Meshtastic Indonesia

👋 Selamat datang di peta node Meshtastic Indonesia! Peta ini menampilkan node-node Meshtastic yang terdeteksi di jaringan MQTT Indonesia.
Fitur
  • Peta menunjukkan node yang mengirimkan posisi valid ke MQTT
  • Posisi harus dalam format tidak terenkripsi atau terenkripsi dengan kunci default
  • Gunakan pencarian untuk menemukan node berdasarkan ID atau nama
  • Arahkan kursor ke node (di desktop) untuk melihat detail dasar
  • Klik node untuk melihat info seperti telemetri dan hubungan antar node
FAQ
Bagaimana cara menambahkan node saya ke peta?
Node Anda, atau node yang mendengar node Anda harus melakukan uplink ke server MQTT kami.
Paket posisi harus tidak terenkripsi, atau terenkripsi dengan kunci default.
Jika node Anda memiliki firmware v2.5 atau lebih baru, Anda harus mengaktifkan "OK to MQTT".
Gunakan detail server MQTT di bawah untuk melakukan uplink ke peta ini.
Server MQTT mana yang harus saya gunakan?
  • Alamat: kemplu.dari.asia
  • Nama Pengguna: idmeshnode
  • Kata Sandi: M3shN0d3
  • Enkripsi Diaktifkan: Ya
  • Keluaran JSON: Tidak
  • TLS Diaktifkan: Ya
Catatan: Kami menyarankan menjalankan node khusus untuk uplink ke peta.
Bagaimana cara menghapus node saya dari peta?
Node yang tidak aktif lebih dari 1 hari otomatis disembunyikan dari peta, tapi datanya tetap tersimpan di database.
Nonaktifkan pelaporan posisi di node Anda untuk mencegah kemunculan kembali.
Gunakan kunci enkripsi khusus sehingga publik tidak bisa melihat data posisi Anda.
Mengapa node saya muncul di tempat yang salah?
Peta publik ini membuat posisi Anda agak samar untuk firmware v2.4 dan lebih lama. Node pada v2.4 dan lebih lama memiliki posisi yang disamarkan hingga 2 digit desimal.
